====== Participer à cette documentation ======

===== Les règles à respecter =====

  * La présente documentation doit uniquement être un support pour l'utilisation du site Ouvretaferme.
  * Aucune donnée personnelle ne doit être enregistrée ni visible sur ce site (aussi bien dans le contenu que dans les médias image, vidéos... partagés).

Tout contenu qui ne respecte pas ces règles sera susceptible d'être supprimé sans préavis.

===== Comment contribuer ? =====

① Commencez par créer un compte pour pouvoir contribuer à cette documentation.

② Si l'article le permet, un bouton "Modifier" apparaît en haut. Cliquez dessus.

③ Apportez vos modifications, cliquez sur "Aperçu" pour vérifier votre contenu, puis cliquez sur "Enregistrer".

===== Quelques astuces =====

==== Pour mettre en forme le texte ====

Utilisez de préférence l'éditeur avancé, cela vous permettra de voir en direct la manière dont votre contenu va s'afficher.

Si vous souhaitez mettre de la couleur sur votre texte, il faut utiliser le code suivant : ''%%<color CODE>Mon texte en couleur</color>%%'' avec CODE : soit le code hexadecimal (par exemple //#000000// pour noir), soit le nom reconnu en anglais (par exemple //black// pour noir)


==== Pour insérer une vidéo Youtube ====

Si vous souhaitez insérer un lien vers une vidéo et que la vidéo s'affiche, tout d'abord **contactez l'équipe pour que la vidéo soit enregistrée sur le compte Youtube de Ouvretaferme**.

Puis choisissez le menu "puzzle" et cliquez sur "bloc d'extension" :

{{:misc:contribute-extension.png}}Dans le bloc avec l'outil, indiquez le code suivant :

''%%{{youtube>ID_DE_LA_VIDEO?full}}%%''

Pour trouver la valeur ID_DE_LA_VIDEO, il faut regarder le lien. Par exemple, pour la vidéo sur la facturation électronique, le lien est [[https://youtu.be/dl3BC-RirkI]]. Donc la valeur ID_DE_LA_VIDEO est //dl3BC-RirkI//.

//?full// sert à indiquer que la vidéo doit prendre toute la largeur d'affichage de l'article.

==== Pour insérer un lien sous forme de bouton ====

Ouvrez le menu "puzzle", et choisissez "Plugin inline". Remplacez le texte fixme par ce code :

''%%<wrap btn--secondary>Texte du bouton</wrap>%%''

Puis sélectionner "Texte du bouton" et ajoutez-y votre lien depuis l'éditeur (icône de chaîne 🔗).

Voici le rendu :

<wrap --btn-secondary>Texte du bouton</wrap>


==== Astuce générale ====

Si vous avez un doute sur comment faire quelque chose, vous pouvez consulter un autre article qui contient ce dont vous avez besoin et regarder comment cela a été réalisé !
